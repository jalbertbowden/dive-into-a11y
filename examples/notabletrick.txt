   Even though it's obvious using a visual browser that this is the main
   content, text-only browsers like Lynx will display the navigation bar
   first, because this page uses tables for layout, and the navigation
   bar comes first in the HTML source, so Lynx displays it first. Search
   engines will also penalize you for this, since it looks like your main
   content is less important than your navigation bar (since the main
   content is further down the page, at least according to the HTML
   source).

   You can solve this problem by using the table trick.
